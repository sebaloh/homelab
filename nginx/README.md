## Installation

For the full installation guide go [here](https://docs.nginx.com/nginx/admin-guide/installing-nginx/installing-nginx-open-source/).

### 1. Download

```sh
sudo apt-get install nginx -y
```

### 2. Check status

```sh
sudo systemctl status nginx
```
